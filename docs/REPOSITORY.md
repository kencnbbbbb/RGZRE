# Repository Notes

В репозитории хранятся только исходные файлы проекта:

- `.cpp` и `.h`;
- `Makefile`;
- `README.md`;
- `.gitignore`;
- документация.

Скомпилированные файлы не хранятся в git:

- `encryption_rgr`;
- объектные файлы `.o`;
- динамические библиотеки `.so` и `.dylib`;
- папка `plugins`;
- тестовые результаты шифрования.

После клонирования нужно выполнить:

```bash
make
./encryption_rgr
```

Команда `make` заново создаст исполняемый файл и динамические библиотеки в `plugins`.
